本项目是一个简单的测试项目，集成了测试中用的几种元素定位、手势操作、数据配置、单元测试、日志文件、测试报告、测试用例、Pageobject设计模式等，主要是对wdj项目的登录功能进行的测试，从csv文件中读取准备好的一些测试用到的用户名、密码，从yaml文件中读取数据的配置信息；

 **框架的大概功能如下：** 
- 业务功能的封装
- 测试用例的封装
- 测试包的管理
- 截图处理
- 日志的获取
- 测试报告的生成
- 数据配置
- 数据驱动

 **测试环境** 
- MacBook Air
- Python3.6
- Appium 1.13.0
- wdj Android版 5.71.21
- 夜神模拟器 Android 4.4.2
